Basic Syntax

Basic Syntax

Lisp code is written as symbolic expressions (s-expressions) using prefix notation.

;; Basic arithmetic
(+ 1 2 3)  ; Addition: 6
(- 10 5)   ; Subtraction: 5
(* 4 3)    ; Multiplication: 12
(/ 15 3)   ; Division: 5

Variables

;; Global variables
(setq x 10)
(defvar *global-var* 100)

;; Local variables
(let ((a 1)
      (b 2))
  (+ a b))

Conditionals

;; If statement
(if (> 5 3)
    "True"
    "False")

;; When and Unless
(when (> 5 3)
  (print "Greater"))

;; Cond - multiple conditions
(cond ((< x 0) "Negative")
      ((> x 0) "Positive")
      (t "Zero"))

Comments

;; Single line comment
#| Multi-line
   comment |#
