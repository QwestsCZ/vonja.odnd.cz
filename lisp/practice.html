Exercise 1: Basic Function

Write a function that calculates the factorial of a number.

;; Example solution:
(defun factorial (n)
  (if (<= n 1)
      1
      (* n (factorial (- n 1)))))

Exercise 2: List Processing

Write a function that reverses a list without using the built-in reverse function.

;; Example solution:
(defun my-reverse (lst)
  (let ((result nil))
    (dolist (elem lst result)
      (push elem result))))
