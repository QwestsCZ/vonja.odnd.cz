Lisp Glossary

S-expression

Symbolic Expression - the basic building block of Lisp code. It consists of atoms or lists enclosed in parentheses, where the first element is typically an operator or function name.

Atom

A basic unit in Lisp that's not a list. This includes numbers, symbols, and strings.

Cons Cell

A fundamental data structure in Lisp that consists of two parts: the car (first element) and the cdr (rest of the list).

Macro

A special type of function that operates on Lisp code itself, allowing for code transformation at compile time.

Lambda

An anonymous function that can be defined and used without giving it a name.
